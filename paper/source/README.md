# Original draft sources

This directory contains the source draft files used to create the LaTeX version
of the paper:

- `original_draft.docx`: Google Docs draft exported as Microsoft Word.
- `original_draft.pdf`: Google Docs draft exported as PDF for visual review.

The LaTeX draft lives in `paper/main.tex`, with appendix material in
`paper/appendix.tex` and references in `paper/references.bib`.
